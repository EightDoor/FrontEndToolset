# 功能
- 开发环境配置
- 开发ide安装

# 运行
- 构建node-gpy
  - 执行 nmp run rebuild

# 打包
- `npm run dist`

  - 配置Prettier  实现代码保存自动格式化

# 依赖
- concurrently 同时运行多条命令
- prettier 代码格式化
- debugout.js 前端日志
- vitejs-plugin-electron electron ESModule使用
- codemirror 代码编辑器
- jsonlint-mod json格式化
- json-to-ts json转换为ts文件
